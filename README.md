#  🌱 IT320-GreenRent-System


## 📌 Overview
GreenRent is a web-based platform designed to facilitate the rental of agricultural equipment in Riyadh.  
It connects equipment owners with renters, making the process of finding, listing, and reserving equipment easier and more efficient.

---

## 🎯 Objectives
- Simplify the process of renting agricultural equipment  
- Help equipment owners list and manage their equipment  
- Allow renters to search and reserve equipment   
- Improve accessibility and efficiency in the agricultural sector  

---

## ⚙️ Features
- 🔍 Filter equipment by type, price, and location  
- 📅 Reserve equipment for specific dates  
- ➕ Add new equipment listings  
- ⭐  Rating and review system  
- 👨‍💼 Admin monitoring 

---
